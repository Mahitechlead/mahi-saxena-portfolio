# 🌐 Mahi Saxena – Portfolio Website

A modern, responsive, and animated **personal portfolio website** built using **HTML, CSS, and JavaScript**, designed to showcase my technical skills, projects, certifications, and achievements as a **Software Engineer** and **B.Tech Computer Science student**.

---

## 🖼️ Live Demo
👉 [mahi-saxena.netlify.app](#)  
---

## 🧩 Features

- 🎨 **Beautiful UI/UX:** Clean, responsive design with glassmorphism and gradient effects  
- 🌙 **Dark/Light Mode:** Smooth toggle with transitions  
- 📱 **Responsive Layout:** Works perfectly on all screen sizes  
- ⚡ **Animated Elements:** Subtle animations for an engaging experience  
- 🧭 **Sticky Navbar:** Smooth scrolling and navigation  
- 💻 **Projects Section:** Showcase of featured works with live links  
- 🎓 **Education, Skills, and Achievements:** Well-structured and visually clean  

---

## 🧠 Sections Overview

| Section | Description |
|----------|-------------|
| 🏠 **Home** | Introduction, tagline, and social links |
| 🎓 **Education** | Academic background with institution details |
| 💻 **Technical Skills** | List of tools, languages, and technologies |
| 🚀 **Projects** | Featured projects with short descriptions |
| 🪪 **Certifications** | Courses and certifications completed |
| 🏆 **Achievements** | Awards and milestones |
| ⚙️ **Footer** | Closing section with contact info |
